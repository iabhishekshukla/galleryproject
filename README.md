Gallery Project

A simple and responsive gallery application built using React.js.
This project fetches images from a public API and displays them in a grid layout with pagination support.

Features

Fetch images from an external API

Responsive gallery layout

Pagination functionality

Fast and lightweight

Clean and simple UI

Technologies Used

React.js

Vite

JavaScript

CSS

 └── assets/

Installation and Setup

Follow the steps below to run the project locally:

1. Clone the repository
git clone https://github.com/your-username/galleryproject.git

2. Open the folder
cd galleryproject

3. Install dependencies
npm install

4. Run the development server
npm run dev

API Used

This project uses the following API to fetch images:

https://picsum.photos/v2/list
